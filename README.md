# Python Buddy AI Assistant

## Overview

Python Buddy AI Assistant is an AI-powered virtual assistant developed using Python and Natural Language Processing (NLP) techniques. The assistant is designed to understand user queries, automate routine tasks, execute commands, retrieve information, and improve productivity through intelligent conversational interactions.

## Problem Statement

Managing repetitive tasks such as note-taking, reminders, information retrieval, and command execution often requires significant manual effort. This project aims to automate these activities through an intelligent assistant capable of understanding natural language and performing user-requested actions efficiently.

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory

```bash
cd Python-Buddy-AI-Assistant
```

3. Create a virtual environment

```bash
python -m venv venv
```

4. Activate the environment

```bash
venv\Scripts\activate
```

5. Install dependencies

```bash
pip install -r requirements.txt
```

6. Run the assistant

```bash
python main.py
```
